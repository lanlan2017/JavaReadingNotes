# 7.7.4 使用MessageFormat处理包含占位符的字符串 #
## `MessageFormat`类 ##

|方法|描述|
|:---|:---|
|`format(String pattern, Object.values)`|返回后面的多个参数值填充前面的`pattern`字符串,其中`pattern`字符串不是正则表达式,而是一个带占位符的字符串。|
