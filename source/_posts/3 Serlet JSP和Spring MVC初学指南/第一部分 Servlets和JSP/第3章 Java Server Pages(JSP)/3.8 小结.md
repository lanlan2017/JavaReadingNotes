## 3.8 小结 ##
`JSP`是构建`Java Web`应用程序的第二种技术，是`Servlet`技术的补充，一个精心设计的`Java Web`应用程序会同时使用`Servlet`和`JSP`两种技术。
在本章中，我们已经学到了`JSP`是如何工作的，以及如何编写`JSP`页面。现在，我们已经知道`JSP`的隐式对象，并能在`JSP`页面中使用**3个语法元素：`指令`、`脚本`和`动作`。**
